# Simple Elementor Image Carousel

Simple Elementor Image Carousel is a lightweight, high-performance WordPress plugin designed to add a customizable and responsive image carousel widget to Elementor. Built with speed and ease of use in mind, it provides a seamless experience for showcasing your images without the bloat.
